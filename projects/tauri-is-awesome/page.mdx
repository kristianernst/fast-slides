# Why teams pick Tauri

<div className="split">
  <div className="col prose-compact">

  - Tiny install size and fast startup.
  - Native OS integration without shipping Chromium.
  - Great fit for utilities, internal tools, and agent workflows.

  </div>
  <div className="col prose-compact">

  ## Stack

  - Rust backend commands
  - Web UI (React/Next/Svelte/etc.)
  - One codebase, native desktop behaviors

  </div>

# Architecture in one minute

<div className="split">
  <div className="col prose-compact">

  1. UI renders in a lightweight webview.
  2. Secure Rust commands handle filesystem + OS features.
  3. IPC keeps the boundary explicit and auditable.

  </div>
  <div className="col prose-compact">

  ## What this gives you

  - Better performance profile
  - Lower memory footprint
  - Easier hardening of critical paths

  </div>
</div>

# FastSlides desktop use case

<div className="split">
  <div className="col prose-compact">

  - Open any folder containing `page.mdx`.
  - Track projects in-app.
  - Resize and collapse sidebar for focus.
  - Render live slide preview inside desktop UI.

  </div>
  <div className="col prose-compact">

  ## Agent-ready workflow

  - Local FastSlides hook API for app state + preview URL.
  - Agent edits MDX, then validates and previews.
  - Screenshot QA loop for fast visual iteration.
